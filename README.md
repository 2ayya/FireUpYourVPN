Fire Up Your VPN
-------

## 在通知中心一键启用 VPN。

### 基础要求：

* Xcode 7+ / Swift 2.0+
* iOS 8+
* 付费苹果开发者账号（用于 VPN）

### 关键特性

alpha 1.0，暂时只支持 CISCO IPSec。

修改配置，TodayViewController 中：

```swift
self.vpnConfig = VPNConfig(server: "www.ooxx.com", username: "user", password: "pass", groupName: "vpn", sharedSecret: "vpn.psk")
```
